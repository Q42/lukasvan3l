    Productcatalogus

    Elk product dat je ooit toevoegde, met per winkel de koppeling (link/artikelnummer) en laatst bekende prijs. Prijzen vult de prijs-agent automatisch; je kunt ze hier ook handmatig invullen. Bij Varuvo verkopen ze per doos: vul dan inhoud in (aantal stuks per doos) zodat de stuksprijs eerlijk vergeleken wordt.

    Nog geen producten.

    Prijzen & back-up

    Alles wordt lokaal in je browser bewaard (localStorage) — er gaat niets naar een server. De prijs-agent (zie README in de repo) draait apart met jouw inloggegevens en produceert een prijzen.json die je hier importeert. Persoonlijke prijzen horen niet in deze publieke repo, dus die gaan via dit importbestand.

    Formaat prijzen.json

    { "opgehaald": "2026-07-08T06:00:00Z", "prijzen": [ { "product": "havermout", // naam of product-id uit de export "shop": "varuvo", // varuvo | ah | vhtg "prijs": 11.95, // prijs zoals in de shop "inhoud": 6, // optioneel: stuks per verpakking/doos "url": "https://…", // optioneel: productlink "omschrijving": "Havermout 6×500g" // optioneel } ] }
